# Imagini LUMÉ

Pune aici cele 6 fotografii. Orice format (`.jpg`, `.png`, `.webp`, `.avif`) —
Astro generează automat variante AVIF + WebP optimizate, cu dimensiuni corecte
(zero CLS). Numele de bază (fără extensie) trebuie să fie exact:

| Nume fișier (bază) | Fotografia | Folosit în |
|---|---|---|
| `hero`     | Ochi close-up, gene lungi, spațiu negativ | Hero + Galerie |
| `lash`     | Aplicare extensii gene (mănuși, pensetă)  | Servicii (Extensii) + Galerie |
| `brow`     | Sprâncene definite, close-up              | Servicii (Sprâncene) + Galerie |
| `skin`     | Femeie relaxată, ten dewy, picături       | Servicii (Îngrijire ten) + Galerie |
| `interior` | Cabină spa, masă, pampas, prosoape        | Galerie |
| `products` | Flat-lay produse skincare (gua sha, roller) | Galerie |

Cât timp un fișier lipsește, în locul lui apare un placeholder cu gradient brand
și numele așteptat — site-ul nu se strică niciodată.
